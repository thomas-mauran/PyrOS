# PyrOS
PyrOS is a small operating system project for learning about kernel development.

### Goals
- [ ] Bootloader
- [ ] Printing to screen
- [ ] Interrupts
- [ ] Keyboard input
- [ ] Memory management
- [ ] User mode
- [ ] Filesystem
- [ ] Multitasking
- [ ] Syscalls
- [ ] Shell
- [ ] neofetch like
- [ ] ls
- [ ] cd
- [ ] mkdir
- [ ] rm
